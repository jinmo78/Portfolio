## 배포 방법

`main` 브랜치 push 시 GitHub Actions(`.github/workflows/deploy.yml`)가 자동으로 GitHub Pages에 배포합니다.

```powershell
cd C:\jinmo\project\Portfolio
git push origin main
```

인증이 필요하면 **사용자가 직접** GitHub PAT를 입력합니다 (에이전트는 토큰을 저장하지 않음).

### 배포 후 확인

1. https://jinmo78.github.io/Portfolio/ 접속
2. GitHub Actions 탭에서 deploy 워크플로우 성공 여부 확인
3. 카카오톡 미리보기 캐시 갱신: https://developers.kakao.com/tool/debugger/sharing
